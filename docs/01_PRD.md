# RUboard - Product Requirements Document (PRD)

## 1. 프로젝트 개요

### 1.1 프로젝트명
**RUboard** - 러시아/CIS 지역 B2B 화장품 발주 관리 플랫폼

### 1.2 배경
에바스코스메틱은 러시아, 카자흐스탄 등 CIS 지역 바이어들에게 화장품을 공급하고 있습니다.
현재 발주 프로세스는 이메일/엑셀 기반으로 운영되어 다음과 같은 문제가 있습니다:
- 발주 현황 파악 어려움
- 수량 조정 시 커뮤니케이션 비효율
- 패킹리스트/인보이스 수동 작성
- 이력 관리 부재

### 1.3 목표
- 발주 프로세스 디지털화
- 실시간 발주 현황 추적
- 자동 문서 생성 (패킹리스트, 인보이스)
- 효율적인 수량 조정 및 협의

---

## 2. 사용자 페르소나

### 2.1 바이어 (Buyer)
- **위치**: 러시아, 카자흐스탄
- **역할**: 화장품 수입/유통
- **니즈**:
  - 제품 카탈로그 열람
  - 발주서 작성 및 제출
  - 발주 진행 상황 확인
  - 최종 가격 확인 (커미션 포함)

### 2.2 매니저 (Manager) - 인덜톤
- **역할**: 중개업체, 커미션 관리
- **니즈**:
  - 발주 확인 및 승인
  - 재고 상황 확인
  - 수량 조정 협의
  - 커미션 관리

### 2.3 공급사 (Supplier) - 에바스
- **역할**: 제품 공급, 패킹, 출고
- **니즈**:
  - 제품 마스터 관리
  - 재고/생산 현황 관리
  - 패킹 진행 및 문서 생성
  - 출고 처리

## 3. 핵심 기능 요구사항

### 3.1 제품 마스터 관리

#### FR-PM-01: 제품 목록 조회
- 브랜드/카테고리별 필터링
- 제품명 검색
- 상태(active/inactive) 필터

#### FR-PM-02: 엑셀 일괄 임포트
- 지원 필드: 품목코드, 품목명(한/영/러), 바코드, 브랜드, 카테고리, 입수량, 단가
- 기존 제품 업데이트 (품목코드 기준)
- 유효성 검사 및 오류 표시

#### FR-PM-03: 가격 관리
- 본사 공급가 (base_price)
- 커미션 (commission)
- 최종가 (final_price = base_price + commission)
- 적용일 기준 이력 관리

### 3.2 발주 관리 (핵심)

#### FR-OR-01: 발주서 생성
- 바이어가 DRAFT 상태로 작성
- 제품 검색 및 선택
- 수량 입력
- 자동 금액 계산

#### FR-OR-02: 발주 확정
- 매니저가 수량 검토/조정
- CONFIRMED 상태로 변경
- 변경 이력 JSONB 저장

#### FR-OR-03: 패킹 진행
- 공급사가 PACKING 상태로 변경
- 패킹리스트 정보 입력
- 중량/CBM 계산

#### FR-OR-04: 출고 완료
- SHIPPED → COMPLETED 상태 전환
- 인보이스 생성

#### FR-OR-05: 발주번호 자동생성
- 형식: {지역코드}-{년도}-{시퀀스}
- 예: KZ-2026-0001

### 3.3 문서 생성

#### FR-DC-01: 패킹리스트 PDF
- 수출자/수하인 정보
- 품목별 수량/중량/CBM
- 총계

#### FR-DC-02: 인보이스 PDF
- 품목별 단가/금액
- 결제 조건
- 총액

### 3.4 보조 기능

#### FR-AX-01: 재고 현황 조회
- ERP 연동 데이터 표시 (읽기 전용)
- 품목별 가용 수량

#### FR-AX-02: 생산 일정 조회
- 기존 데이터 마이그레이션
- 예정일/수량 표시

## 4. 비기능 요구사항

### 4.1 성능
- 페이지 로딩: 3초 이내
- 엑셀 임포트: 1000건 기준 30초 이내

### 4.2 보안
- Supabase Auth 기반 인증
- 역할 기반 접근 제어 (RLS)

### 4.3 호환성
- 최신 Chrome, Safari, Edge 지원
- 반응형 디자인 (데스크톱 우선)

### 4.4 언어
- UI: 한국어 (1차)
- 향후 러시아어/영어 지원 고려

## 8. 용어 정의

| 용어 | 정의 |
|------|------|
| 품목코드 (product_code) | 제품 고유 식별자 (예: PDSP009) |
| 입수량 (pcs_per_carton) | 박스(카톤)당 개수 |
| 본사가 (base_price) | 에바스 공급가격 |
| 커미션 (commission) | 인덜톤 마진 |
| 최종가 (final_price) | 바이어 결제가 (본사가+커미션) |
| 패킹리스트 | 출고 시 품목/수량/중량 명세서 |
| 인보이스 | 상업송장 (금액 명시) |
